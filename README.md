## üìå Project Overview

EEG signals represent electrical activity in the brain captured over time. Using deep learning techniques like CNNs, we can extract meaningful spatial and temporal features directly from processed EEG data for classification tasks such as mental state decoding and brain‚Äëcomputer interfaces (BCIs).

This project focuses on training a CNN model on EEG signal data and evaluating its performance through training history and visual inspection.
